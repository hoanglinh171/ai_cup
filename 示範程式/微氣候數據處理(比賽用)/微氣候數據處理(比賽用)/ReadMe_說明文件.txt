程式名稱 : 天氣小助手

程式目的 : 此程式用於查看、切割太陽能周邊微氣候數據

程式環境 : Dot Net 8.0 for Windows

注意事項 :

1. 請點選程式執行檔開始使用程式，並建議每次都使用系統管理員身分執行。

2. 使用前請先觀看教學影片以及相關文件

3. 資料都包含在數據庫裡，但數據庫裡的CSV檔案並未包含標題。

4. 如有新的檔案要加入，直接拖曳進數據庫的資料夾即可，並請注意副檔名應為CSV。

5. 格式為裝置代號、時間、風速、大氣壓力、溫度、濕度、光照度、發電量，請勿更動格式。

6. 程式僅為輔助使用者查看資料，請勿以程式的資料當作基準，應再三檢查。

7. 程式所生成之文件都未包含標題，再匯入時也請注意把標題移除。

Please review the following instructions:
========================================================
Program Name: Weather Assistant

Program Purpose: This program is used for viewing and segmenting microclimate data around solar energy installations.

Program Environment: Dot Net 8.0 for Windows

Important Notes:

1. Please click on the program executable file to start using the program. It is recommended to run it as an administrator each time.

2. Before using, please watch the tutorial video and review the related documentation.

3. All data is contained in the database, but the CSV files in the database do not include headers.

4. If you need to add new files, simply drag them into the database folder, ensuring that the file extension is CSV.

5. The format is: device code, time, wind speed, atmospheric pressure, temperature, humidity, illumination, power generation. Please do not alter the format.

6. The program is intended to assist users in viewing the data. Do not rely solely on the program’s data as the final reference; double-check the information.

7. The documents generated by the program do not include headers. Please also ensure to remove the headers when importing data.
